## 📍 模块一：测量（Measurement）

### 功能描述
实时GPS定位和项目坐标显示，是核心工作界面。

### 主要功能区

#### 1. GPS定位信息卡片
显示设备实时定位数据：
- **纬度**：当前设备纬度坐标
- **经度**：当前设备经度坐标
- **精度**：GPS定位精度值（单位：米）
- **海拔**：当前海拔高度（单位：米）

**说明**：精度值越小越好（推荐 < 5米）

#### 2. 项目坐标卡片
显示自动转换后的项目坐标：
- **X坐标**：项目坐标系X轴位置（米）
- **Y坐标**：项目坐标系Y轴位置（米）
- 实时显示坐标自动转换状态

#### 3. 操作按钮
- **开始定位**：开启GPS定位，持续更新位置
- **停止定位**：关闭GPS定位，暂停位置更新
- **刷新数据**：立即获取最新的定位坐标

### 使用流程
1. 进入测量页面
2. 点击 **开始定位** 按钮
3. 等待GPS锁定（精度稳定）
4. 查看自动转换的项目坐标
5. 需要新数据时点击 **刷新数据**

## 📷 模块二：AR视图（AR View）

### 功能描述
增强现实实景显示，将工程图纸和3D模型叠加到相机实景中。

### 主要功能区

#### 1. AR相机视图区域
- 绿色边框：表示AR识别框
- 绿色网格线：坐标辅助线（可关闭）
- 建筑物📑 实心：代表3D模型
- "正常状态" 或 "偏差超标警告"：实时状态提示

#### 2. 坐标显示条
显示当前AR中的三维坐标：
```
X: 1024.5m  Y: 2048.3m  Z: 45.2m
```

#### 3. 控制面板（下方）

**模型透明度** - 滑块控制
- 调节范围：0% ~ 100%
- 0% = 完全透明（看不见模型）
- 100% = 完全不透明（模型最清晰）

**网格辅助线** - 开关控制
- ON：显示绿色网格线，便于对齐
- OFF：隐藏网格线，减少干扰

**偏差检测** - 开关控制
- ON：启用偏差监测，当超标时显示红色警告框
- OFF：关闭检测，始终显示正常状态

**截图保存** - 按钮
- 保存当前AR视图为image文件

### 使用场景
1. 在工地现场对准建筑位置
2. 调节透明度以对齐参考线
3. 启用偏差检测进行实时监测
4. 遇到异常立即截图保存作为证据

## 💾 模块三：离线管理（Offline Management）

### 功能描述
管理本地离线数据和云端同步。

### 主要功能区

#### 1. 存储统计卡片
- **存储使用比例**：进度条显示
- **使用容量**：例 "6.5GB / 10GB"
- **上次同步时间**：最后一次数据同步的时间戳

#### 2. 立即同步按钮
- **状态1**："立即同步" - 点击开始同步数据
- **状态2**："同步中..." - 正在进行同步操作

#### 3. 离线项目列表
每个项目卡片包含：

**项目名称和同步状态**
- 项目名称：如 "北京中关村项目"
- 同步状态标签：
  - 绿色 **已同步**
  - 橙色 **未同步**

**项目详情行**
| 数据项 | 示例 |
|--------|------|
| 大小 | 245MB |
| 数据点 | 1250 |
| 最后同步 | 14:30 |

#### 4. 操作按钮
- **添加项目**：下载新的工程项目到本地
- **清理缓存**：删除缓存数据以释放存储空间

### 使用场景
**无网环境前准备**：
1. 进入离线管理
2. 查看项目同步状态
3. 对未同步项目点击 **立即同步**
4. 等待同步完成后离线作业

**存储清理**：
1. 当接近10GB时点击 **清理缓存**
2. 删除不需要的数据

## 📋 模块四：记录（Records）

### 功能描述
查看历史测量记录，分析异常数据。

### 主要功能区

#### 1. 统计信息卡片
展示三个关键指标：

| 指标 | 说明 |
|------|------|
| 总记录数 | 所有历史记录总数 |
| 异常记录 | 偏差超标的记录数 |
| 今日记录 | 当天新增的记录数 |

#### 2. 筛选按钮
- **全部**：显示所有记录
- **异常**：仅显示偏差超标的记录
- **今日**：仅显示当天的记录

#### 3. 记录列表

每条记录卡片显示：

**记录头部**
- 项目名称
- 测量时间：日期 + 时刻
- 状态标签：
  - 🟢 **正常** (绿色)
  - 🟡 **注意** (橙色, 偏差0.2-0.5mm)
  - 🔴 **异常** (红色, 偏差>0.5mm)

**坐标信息**
```
X: 1024.5m, Y: 2048.3m
```

**偏差值**
- 单位：毫米（mm）
- 示例："0.48 mm"

#### 4. 操作按钮
- **导出数据**：将所有记录导出为CSV/Excel格式
- **分析报告**：生成统计分析报告

### 使用场景
**查找异常**：
1. 点击 **筛选 - 异常**
2. 进行集中排查和处理

**获取支持数据**：
1. 点击 **导出数据**
2. 提供给团队主管或云端平台

## 关键操作流程

### 工程测量完整流程
```
1. 进入 测量 页面
   ↓
2. 点击 开始定位 等待GPS锁定
   ↓
3. 查看 项目坐标 确认位置正确
   ↓
4. 在 AR视图 中验证实际位置
   ↓
5. 如有偏差异常，系统自动记录
   ↓
6. 进入 记录 页面查看历史
   ↓
7. 点击 导出数据 或 分析报告
```

### 离线作业准备
```
1. 进入 离线管理
   ↓
2. 检查项目同步状态
   ↓
3. 对需要的项目点击 立即同步
   ↓
4. 进入 我的 → 工作模式 切换离线
   ↓
5. 即可在无网环境作业，数据本地保存
```

## 数据说明

### 坐标系统
- **GPS坐标**：纬度/经度（WGS84标准）
- **项目坐标**：X/Y轴（工程项目坐标系）
- **自动转换**：基于后方交汇算法

### 偏差判断
| 偏差值 | 状态 | 说明 |
|--------|------|------|
| 0 ~ 0.1mm | 🟢 正常 | 误差在允许范围内 |
| 0.1 ~ 0.5mm | 🟡 注意 | 接近限值，需要关注 |
| > 0.5mm | 🔴 异常 | 超标，需要立即处理 |

### 精度等级
| GPS精度 | 等级 | 是否可用 |
|----------|------|---------|
| 1-5m | 优 | ✅ 可用 |
| 5-10m | 良 | ⚠️ 可用(需谨慎) |
| > 10m | 差 | ❌ 不可用 |

## 快速问题排查

**Q: GPS无法定位？**
- 检查设备位置权限是否授予
- 确认在室外开阔环境
- 稍等30秒让GPS初始化

**Q: 坐标转换不正确？**
- 检查当前定位精度是否 < 5米
- 尝试在 我的 中点击自动校准

**Q: 如何清理缓存？**
- 进入 离线管理 → 清理缓存 按钮

**Q: 如何导出数据？**
- 进入 记录 → 导出数据 按钮
